О проекте
Order Management System for Cafe — это информационная система для автоматизации процесса обработки заказов в небольшом кафе или ресторане. Проект фокусируется на ключевом бизнес-процессе заведения — от момента прихода гостя до полного расчета.

Основная цель проекта — создание удобного инструмента для официантов и администраторов, который ускоряет обслуживание, исключает ошибки в расчетах и обеспечивает прозрачный учет продаж.

Ключевые возможности
Управление меню: Ведение актуального меню с категориями блюд, ценами, описанием.

Быстрое оформление заказов: Интуитивный интерфейс рабочего места официанта с возможностью:

Выбора стола (или отметки "навынос")

Добавления блюд из меню в заказ

Изменения количества позиций

Комментирования заказа (пожелания гостя)

Работа с заказами: Просмотр открытых заказов, редактирование, объединение/разделение заказов по столам.

Печать на кухню: Автоматическое формирование пречека для кухни при подтверждении заказа (эмуляция вывода на печать).

Расчет гостя: Формирование счета, возможность разделения счета на несколько гостей/способов оплаты.

Прием платежей: Фиксация оплаты наличными, банковской картой, переводом.

Закрытие заказа: Перевод заказа в статус "оплачен", освобождение стола.

История заказов: Просмотр архива закрытых заказов за смену/день.

Экран администратора: Управление меню, просмотр текущей выручки, контроль действий официантов.
